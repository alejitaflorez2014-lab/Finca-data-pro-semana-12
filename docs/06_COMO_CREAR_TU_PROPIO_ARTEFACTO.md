# Cómo crear tu propio artefacto

La actividad evaluada no consiste solo en ejecutar este proyecto. Debes crear o adaptar una mini-solución propia.

## Opción 1: Cambiar el contexto

Puedes adaptar el proyecto a otro caso:

- Tienda escolar.
- Inventario de biblioteca.
- Registro de asistencia.
- Control de riego.
- Producción de huevos.
- Gastos de transporte.

## Opción 2: Cambiar los datos

Crea un nuevo archivo CSV con tus propias columnas.

Ejemplo para tienda:

```csv
fecha,producto,cantidad_vendida,precio_unitario
2026-05-18,arepa,20,2500
2026-05-19,jugo,15,3000
```

## Opción 3: Agregar una alerta

Puedes agregar una alerta como:

```text
Si la producción baja de 20 litros, mostrar advertencia.
```

## Opción 4: Mejorar el reporte

Puedes agregar:

- Conclusión automática.
- Recomendación.
- Comparación con la semana anterior.
- Porcentaje de crecimiento.

## Opción 5: Personalizar el dashboard

Puedes cambiar:

- Título.
- Colores.
- Métricas.
- Textos.
- Secciones.

## Qué debe tener tu README

Tu README debe responder:

1. ¿Qué problema resuelve?
2. ¿Qué archivo usa como entrada?
3. ¿Qué proceso realiza?
4. ¿Qué archivos genera?
5. ¿Cómo se ejecuta?
6. ¿Qué aprendiste?
