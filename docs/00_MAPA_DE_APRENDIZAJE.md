# Mapa de aprendizaje de la actividad

Esta actividad está diseñada en tres niveles: comprensión, ejecución y creación.

## Nivel 1: Comprensión

El estudiante identifica conceptos fundamentales:

- Script.
- Automatización.
- Respaldo.
- Reporte.
- Archivo CSV.
- Validación de datos.
- Dashboard.
- Bitácora técnica.

## Nivel 2: Ejecución guiada

El estudiante ejecuta un proyecto ya construido para observar cómo funciona un flujo real:

```text
CSV de entrada -> Validación -> Análisis -> Reporte -> Respaldo -> Dashboard -> Bitácora
```

## Nivel 3: Creación del artefacto

El estudiante adapta el proyecto o crea una mini-solución propia. La entrega debe demostrar que comprende el flujo y que puede modificarlo con autonomía.

## Relación con la actividad evaluada

La evidencia solicitada es: **archivo(s) del proyecto + README breve**. Por eso este proyecto no se limita a un cuestionario; exige que el estudiante entregue algo ejecutable y explicable.
