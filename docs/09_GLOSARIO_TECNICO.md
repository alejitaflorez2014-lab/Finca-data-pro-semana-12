# Glosario técnico

## Script

Programa corto que automatiza una tarea específica.

## Automatización

Proceso de hacer que una tarea se ejecute con mínima intervención humana.

## CSV

Archivo de texto que organiza datos separados por comas.

## Respaldo

Copia de seguridad de un archivo importante.

## Reporte

Documento que resume datos y resultados.

## Dashboard

Vista visual de indicadores principales.

## Validación

Proceso de revisar si los datos son correctos antes de usarlos.

## Log

Registro técnico de eventos ocurridos durante la ejecución de un programa.

## Terminal

Herramienta donde se escriben comandos para ejecutar programas.

## README

Archivo que explica qué hace un proyecto, cómo se ejecuta y cómo se entrega.

## Módulo

Archivo de código que agrupa funciones relacionadas.

## Función

Bloque de código reutilizable que realiza una tarea específica.

## Argumento de terminal

Dato que se envía al programa al ejecutarlo.

Ejemplo:

```bash
python src/main.py --data data/produccion_reto.csv
```
